## 基于Android的健身信息管理系统

本系统使用Tomcat作为服务器，数据库采用MySQL。

Tomcat作为Servlet容器	JDBC连接MySQL



三种用户类型：管理员、教练、会员

输入账号密码后，进入对应用户类型的主界面
